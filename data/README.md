# Data

이 폴더에는 프로젝트 분석 과정에서 생성되거나 코드 실행에 필요한 데이터 파일을 저장한다.

## 파일 목록

### `youth_single_household_share_2023_2025.csv`

서울시 행정동별 청년 1인세대 비중을 산출한 결과 데이터이다.

### 지표 목적

행정동별 전체 세대 중 청년 1인세대가 차지하는 비중을 계산하여, 청년 1인세대가 상대적으로 많이 분포하는 지역을 비교하기 위해 사용한다.

### 분석 기준

- 분석 지역: 서울특별시
- 공간 단위: 행정동
- 분석 기간: 2023~2025년
- 청년 연령 기준: 20~39세
- 관측 연도 수: 3개 연도

### 산식

```text
청년 1인세대 비중(%)
= 청년 1인세대수 ÷ 전체 세대수 × 100
```

### 주요 컬럼

| 컬럼명 | 설명 |
|---|---|
| `행정기관코드` | 행정동을 식별하기 위한 행정기관 코드 |
| `시도` | 시도명 |
| `자치구` | 자치구명 |
| `행정동` | 행정동명 |
| `관측연도수` | 지표 산출에 사용한 연도 수 |
| `관측시작연도` | 관측 시작 연도 |
| `관측종료연도` | 관측 종료 연도 |
| `전체세대수` | 지표 산출에 사용된 행정동별 전체 세대수 |
| `청년1인세대수` | 20~39세 청년 1인세대수 |
| `청년1인세대비중` | 전체 세대 중 청년 1인세대가 차지하는 비율 |
| `저표본_여부` | 청년 1인세대수가 적어 지표 해석에 주의가 필요한 지역 표시 |

### 관련 노트북

아래 순서로 실행한다.

1. `notebooks/01_preprocess_youth_single_household.ipynb`
   - 연도·자치구별 주민등록 1인세대 원본 파일을 불러온다.
   - 20~39세 청년 1인세대수를 추출한다.
   - 행정동 단위로 데이터를 정리하고 연도별 결과를 통합한다.

2. `notebooks/02_calculate_youth_single_household_share.ipynb`
   - 청년 1인세대수와 전체 세대수 데이터를 결합한다.
   - 행정동별 청년 1인세대 비중을 계산한다.
   - 관측기간 및 표본 주의 여부를 생성한다.
   - 최종 CSV 파일을 저장한다.

### 해석 시 유의사항

- 본 지표는 행정동 면적당 밀도가 아니라 전체 세대 중 청년 1인세대가 차지하는 구성비이다.
- 주민등록 세대 자료를 기반으로 하므로 실제 생활인구 또는 유동인구와는 차이가 있을 수 있다.
- 청년 1인세대수가 적은 지역은 비율 변동성이 클 수 있으므로 표본 주의 여부를 함께 확인해야 한다.
